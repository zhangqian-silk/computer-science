# Cache Component

缓存组件用于提供统一缓存能力，覆盖本地缓存、分布式缓存与多级缓存协同策略。

---

## 组件边界

- 负责：缓存读写策略、多级缓存协同、热点治理、一致性与可观测。
- 不负责：业务主存设计、业务事务补偿编排。

---

## 核心知识点清单

- 缓存模型：KV、对象缓存、查询结果缓存。
- 淘汰策略：LRU、LFU、TTL、分层淘汰。
- 读写模式：Cache Aside、Write Through、Write Back。
- 热点治理：热点 key 识别、打散、隔离。
- 一致性治理：双写一致、失效策略、回源控制。
- 可观测性：命中率、回源率、穿透率、雪崩告警。
