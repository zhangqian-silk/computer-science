# Rate Limiter Component

限流器组件用于提供统一流量治理能力，目标是在高并发场景下保护核心依赖并控制系统抖动。

---

## 组件边界

- 负责：流量准入判定、限流策略执行、限流指标输出。
- 不负责：业务重试编排、业务降级页面渲染、业务风控规则。

---

## 核心知识点清单

- 限流算法：固定窗口、滑动窗口、漏桶、令牌桶。
- 限流粒度：IP、用户、接口、租户、渠道、地域。
- 限流位置：网关层、服务层、下游依赖层。
- 分布式实现：Redis + Lua 原子判定、时钟偏差处理。
- 配置能力：动态开关、阈值热更新、白名单/黑名单。
- 观测指标：通过率、拒绝率、误伤率、恢复时间。
