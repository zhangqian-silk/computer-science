# Search Component

搜索组件用于提供统一检索能力，覆盖索引构建、召回、排序与查询服务。

---

## 组件边界

- 负责：索引构建、查询解析、召回排序、检索可观测与故障降级。
- 不负责：业务侧推荐策略编排、运营规则审批与素材生命周期管理。

---

## 核心知识点清单

- 索引模型：倒排索引、正排存储、字段权重与分词策略。
- 检索链路：查询改写、过滤、召回、粗排、精排。
- 索引更新：全量构建、增量刷新、近实时写入与回滚。
- 一致性治理：索引版本栅栏、双写比对、读写隔离。
- 性能治理：分片路由、缓存协同、慢查询熔断与退化。
- 可观测性：QPS、P95 延迟、召回率、索引延迟、失败率。
