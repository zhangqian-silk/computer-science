## 0. 符号约定与公式索引

统一使用以下符号：

| 符号 | 含义 |
| --- | --- |
| \(x\) | 输入向量 |
| \(W^{(l)}, b^{(l)}\) | 第 \(l\) 层的权重矩阵与偏置向量 |
| \(z^{(l)}\) | 第 \(l\) 层线性变换结果 |
| \(a^{(l)}\) | 第 \(l\) 层激活输出 |
| \(f(\cdot)\) | 隐藏层激活函数 |
| \(g(\cdot)\) | 输出层输出函数 |
| \(y\) | 真实标签或真实目标 |
| \(\hat{y}\) | 模型预测输出 |
| \(\mathcal{L}\) | 损失函数值 |
| \(\theta\) | 模型参数集合 |
| \(\eta\) | 学习率 |
| \(\delta^{(l)}\) | 第 \(l\) 层的局部误差信号 |

本文核心公式索引：

1. 线性变换：
\[
z^{(l)}=W^{(l)}a^{(l-1)}+b^{(l)}
\]

2. 激活变换：
\[
a^{(l)}=f(z^{(l)})
\]

3. 输出层映射：
\[
\hat{y}=g(z^{(L)})
\]

4. 经验风险最小化目标：
\[
\min_{\theta}\ \mathbb{E}_{(x,y)\sim\mathcal{D}}[\mathcal{L}(\hat{y},y)]
\]

5. 梯度下降更新：
\[
\theta \leftarrow \theta-\eta \nabla_{\theta}\mathcal{L}
\]

6. sigmoid 输出：
\[
\sigma(z)=\frac{1}{1+e^{-z}}
\]

7. softmax 输出：
\[
\hat{y}_i=\frac{e^{z_i}}{\sum_j e^{z_j}}
\]

8. 均方误差：
\[
\mathcal{L}_{MSE}=\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i-y_i)^2
\]

9. 交叉熵损失：
\[
\mathcal{L}_{CE}=-\sum_i y_i\log \hat{y}_i
\]

10. 反向传播链式法则：
\[
\frac{\partial \mathcal{L}}{\partial W}
=
\frac{\partial \mathcal{L}}{\partial \hat{y}}
\frac{\partial \hat{y}}{\partial z}
\frac{\partial z}{\partial W}
\]

## 1. 定义与直观理解

激活函数、输出函数、损失函数、前向传播与反向传播并不是彼此独立的零散概念，而是神经网络中一条连续计算链路的不同环节。对一个监督学习模型而言，训练过程可以概括为：

\[
x \rightarrow z \rightarrow a \rightarrow \hat{y} \rightarrow \mathcal{L} \rightarrow \nabla_{\theta}\mathcal{L} \rightarrow \theta'
\]

其中，输入 \(x\) 先经过若干层线性变换与非线性变换，得到预测结果 \(\hat{y}\)；预测结果与真实标签 \(y\) 的差异由损失函数 \(\mathcal{L}\) 度量；随后，通过反向传播计算参数梯度，并由优化器完成参数更新。

若使用一个极简的二分类示例，上述链路可以理解为：

- 输入是一条样本表示 \(x\)；
- 网络对该样本计算一个类别概率 \(\hat{y}\)；
- 若真实标签为正类而 \(\hat{y}\) 较小，则损失较大；
- 损失经反向传播转化为梯度，参数朝着提高正确类别概率的方向更新。

因此，这组机制讨论的并不是某一种模型结构，而是神经网络如何产生预测、如何度量误差以及如何学习参数。

## 2. 核心机制或计算过程

### 2.1 从线性变换到激活输出

神经网络中最基本的层计算形式为：

\[
z^{(l)}=W^{(l)}a^{(l-1)}+b^{(l)}
\]

\[
a^{(l)}=f(z^{(l)})
\]

其中，\(z^{(l)}\) 是线性变换结果，\(a^{(l)}\) 是激活后的输出。若网络仅由线性变换构成，则多层复合仍可化为单层线性映射：

\[
W_2(W_1x+b_1)+b_2 = W'x+b'
\]

因此，激活函数的核心作用是引入非线性表达能力。

### 2.2 激活函数

激活函数（activation function）是作用于 \(z\) 的非线性映射。常见形式如下：

| 名称 | 公式 | 输出范围 | 常见特点 |
| --- | --- | --- | --- |
| sigmoid | \(\sigma(z)=\frac{1}{1+e^{-z}}\) | \((0,1)\) | 输出平滑，常用于二分类输出层 |
| tanh | \(\tanh(z)\) | \((-1,1)\) | 零中心，早期隐藏层中较常见 |
| ReLU | \(\max(0,z)\) | \([0,+\infty)\) | 计算简单，深层网络中广泛使用 |
| Leaky ReLU | \(\max(\alpha z,z)\) | 实数域 | 缓解 ReLU 负区间梯度为零的问题 |
| GELU | \(z\Phi(z)\) 的近似形式 | 实数域 | Transformer 系列模型中常见 |

激活函数既影响前向传播时的数值分布，也影响反向传播时的梯度传播。sigmoid 与 tanh 在输入绝对值较大时容易进入饱和区，导数接近于零；ReLU 在正区间导数恒为 1，但在负区间导数为 0；GELU 提供更平滑的非线性，在大规模语言模型中使用较多。

### 2.3 输出函数

输出函数（output function）位于网络最后一层，用于将内部表示转化为任务所需的预测形式。常见对应关系如下：

| 任务 | 输出形式 | 输出函数 |
| --- | --- | --- |
| 回归 | 实数值 | 线性输出 |
| 二分类 | 单个概率 | sigmoid |
| 多分类 | 类别概率分布 | softmax |
| 多标签分类 | 多个独立概率 | 多个 sigmoid |
| 语言模型 | 词表上的概率分布 | softmax |

在多分类任务中，输出层常先产生未归一化打分：

\[
z_i=W_i a+b_i
\]

再通过 softmax 得到概率分布：

\[
\hat{y}_i=\frac{e^{z_i}}{\sum_j e^{z_j}}
\]

这里的 \(z_i\) 常称为 logits，表示各类别的原始打分。softmax 将这些打分归一化为总和为 1 的概率分布。

### 2.4 损失函数

损失函数（loss function）用于刻画预测结果与真实目标之间的差异：

\[
\mathcal{L}=\mathcal{L}(\hat{y},y)
\]

不同任务通常对应不同损失形式：

| 任务 | 常见损失 | 数学含义 |
| --- | --- | --- |
| 回归 | MSE、MAE | 度量预测值与真实值之间的数值偏差 |
| 二分类 | Binary Cross Entropy | 度量真实类别概率的偏差 |
| 多分类 | Cross Entropy | 度量真实类别是否分配到足够概率 |
| 序列生成 | 逐位置交叉熵 | 对每个时间步的正确输出进行监督 |

对于 one-hot 多分类任务，交叉熵损失为：

\[
\mathcal{L}_{CE}=-\sum_i y_i\log \hat{y}_i
\]

若真实类别为第 \(k\) 类，则可化为：

\[
\mathcal{L}_{CE}=-\log \hat{y}_k
\]

这一形式直接对应“提高真实类别概率”的训练目标。

### 2.5 前向传播与反向传播

前向传播（forward propagation）是指在参数固定时，自输入层向输出层逐层计算，最终得到预测值与损失值的过程。对两层网络而言：

\[
z^{(1)}=W^{(1)}x+b^{(1)}, \quad a^{(1)}=f(z^{(1)})
\]

\[
z^{(2)}=W^{(2)}a^{(1)}+b^{(2)}, \quad \hat{y}=g(z^{(2)})
\]

\[
\mathcal{L}=\mathcal{L}(\hat{y},y)
\]

反向传播（backpropagation）是利用链式法则计算损失对各层参数梯度的过程。若记输出层局部误差信号为：

\[
\delta^{(2)}=\frac{\partial \mathcal{L}}{\partial z^{(2)}}
\]

则有：

\[
\frac{\partial \mathcal{L}}{\partial W^{(2)}}=\delta^{(2)}(a^{(1)})^\top
\]

\[
\delta^{(1)}=((W^{(2)})^\top\delta^{(2)})\odot f'(z^{(1)})
\]

\[
\frac{\partial \mathcal{L}}{\partial W^{(1)}}=\delta^{(1)}x^\top
\]

前向传播负责计算中间量、输出与损失；反向传播负责将损失转换为参数梯度。

## 3. 在模型中的使用方式

这组机制通常不会单独作为完整模型出现，而是作为神经网络中的基础组成部分存在。

### 3.1 在不同模型中的位置

| 模型类型 | 激活函数 | 输出函数 | 损失函数 | 传播方式 |
| --- | --- | --- | --- | --- |
| 前馈网络 | 隐藏层非线性变换 | 分类或回归输出 | 交叉熵 / MSE | 标准前向与反向传播 |
| 卷积网络 | 卷积层后的非线性变换 | 分类输出层 | 交叉熵 | 卷积计算图上的梯度传播 |
| RNN / LSTM | 状态更新中的非线性与门控函数 | 序列输出层 | 序列损失 | 时间展开后的反向传播 |
| Transformer | 前馈子层中的 GELU / ReLU | softmax 输出层 | token 级交叉熵 | 深层残差结构上的梯度传播 |

### 3.2 训练阶段的使用方式

在训练阶段，这组机制按照固定顺序协同工作：

1. 输入样本进入网络；
2. 各层执行线性变换与激活变换；
3. 输出层生成预测结果；
4. 损失函数根据预测与标签计算误差；
5. 反向传播计算各层参数梯度；
6. 优化器依据梯度更新参数。

对应的伪代码为：

```text
TRAIN-STEP(batch, θ)
	ŷ ← FORWARD(batch.x, θ)
	L ← LOSS(ŷ, batch.y)
	grads ← BACKWARD(L, θ)
	θ ← UPDATE(θ, grads)
	return L, θ
```

### 3.3 推理阶段的使用方式

推理阶段只保留前向计算，不再执行损失计算之后的梯度回传与参数更新。其流程通常为：

1. 输入样本；
2. 逐层计算激活输出；
3. 在输出层得到概率、类别、连续值或词表分布；
4. 根据任务规则生成最终输出。

例如，二分类任务中输出层经过 sigmoid 后得到类别概率；语言模型中输出层经过 softmax 后得到下一个 token 的概率分布。

## 4. 一个最小推演例子

考虑一个三分类任务。某个样本经过隐藏层后，输出层产生 logits：

\[
z=[2.0,\ 1.0,\ 0.1]
\]

真实标签为第二类，即：

\[
y=[0,1,0]
\]

这一样本在训练中的最小推演过程如下：

| 步骤 | 结果 | 含义 |
| --- | --- | --- |
| 1 | 输出层给出 \(z=[2.0,1.0,0.1]\) | 三个类别的原始打分 |
| 2 | \(\hat{y}=\mathrm{softmax}(z)\) | 将打分转换为概率分布 |
| 3 | \(\hat{y}\approx[0.659,0.242,0.099]\) | 模型当前更偏向第一类 |
| 4 | \(\mathcal{L}=-\log 0.242\) | 真实类别是第二类，损失较大 |
| 5 | 反向传播计算梯度 | 提高第二类打分，压低错误高分项 |
| 6 | 参数更新后再次前向计算 | 第二类概率应上升，损失应下降 |

若把 logits 改为：

\[
z=[0.2,\ 2.0,\ 0.1]
\]

则 softmax 后第二类概率会明显提高，交叉熵损失相应下降。这个例子说明，输出函数负责将打分转成概率，损失函数负责度量错误大小，反向传播负责把错误转化为参数更新方向。

## 5. 常见变体、优势与局限

### 5.1 常见变体

这组基础机制内部存在多种常见选择：

- 激活函数可取 sigmoid、tanh、ReLU、Leaky ReLU、GELU；
- 输出层可取线性输出、sigmoid、softmax；
- 损失函数可取 MSE、MAE、二元交叉熵、多类交叉熵；
- 反向传播可结合不同优化器，如 SGD、Momentum、Adam。

这些变体并不改变整体训练链路，但会显著影响训练稳定性、数值范围与收敛速度。

### 5.2 优势

这组机制之所以成为神经网络训练的共同基础，主要原因包括：

- 可以把复杂函数分解为多层可微映射；
- 可以用统一的损失函数形式表达多种任务目标；
- 可以通过链式法则高效计算所有参数的梯度；
- 可以在不同网络结构之间复用同一套训练逻辑。

### 5.3 局限与训练难点

| 问题 | 产生原因 | 常见缓解方式 |
| --- | --- | --- |
| 梯度消失 | 多层局部导数持续偏小 | ReLU 类激活、残差连接、归一化 |
| 梯度爆炸 | 多层梯度连乘过大 | 梯度裁剪、参数初始化、归一化 |
| 神经元失活 | ReLU 在负区间长期导数为零 | Leaky ReLU、PReLU |
| 数值不稳定 | softmax、对数与指数运算尺度过大 | logits 版本损失、稳定实现 |
| 收敛缓慢 | 学习率设置不当或损失地形复杂 | Adam、学习率调度、批标准化 |

这些问题不依赖某一个具体模型，而是广泛存在于可微神经网络的训练过程中。

## 6. 与相关机制或模型的关系

这组机制与其他常见内容之间的关系可以概括如下：

| 主题 | 关系 |
| --- | --- |
| attention | attention 模块本身也依赖线性变换、softmax 与反向传播训练 |
| 位置编码 | 位置编码负责补充位置信息，但其参数学习仍依赖损失函数与反向传播 |
| embedding | embedding 提供输入表示，其参数更新也依赖梯度传播 |
| RNN / LSTM | 反向传播在序列模型中表现为时间展开后的梯度传播 |
| Transformer | 前馈层中的 GELU、输出层 softmax 与交叉熵训练都依赖这组机制 |

从范围上看，激活函数、输出函数、损失函数与前向、反向传播属于更基础的训练机制；attention、位置编码、门控结构等则是在这一基础之上构造出的更具体模块。

---

## 7. 小结

激活函数、输出函数、损失函数、前向传播与反向传播构成了神经网络训练中的基础机制链路。

- 激活函数决定中间层如何形成非线性表示；
- 输出函数决定最终预测结果的数学形式；
- 损失函数决定训练过程中优化的目标；
- 前向传播负责从输入计算到预测与损失；
- 反向传播负责将损失转换为参数梯度。

这组机制并不对应某一个具体模型，而是几乎所有可微神经网络共享的训练基础。
