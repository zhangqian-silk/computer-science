# NLP 层：从历史主线进入现代方法地图

`nlp/` 目录关注的不是某一个单独模型，而是自然语言处理方法如何沿着历史主线不断演化。

这一层主要回答：

- NLP 的主干问题是如何一步步改变的；
- 统计方法、表示学习、序列模型、注意力与预训练之间是什么关系；
- 为什么现代方法会组织成今天这样的知识地图。
